# Developer Portfolio

This project is my personal portfolio site and my resume presented as a web app.

It is also an ongoing experiment for me to train and improve how I work with AI-assisted coding tools while building real features and refining UI details.

## What This Project Is

- A public-facing portfolio to showcase experience, skills, and projects
- A resume-style narrative in a more modern, interactive format
- A hands-on sandbox for practicing AI-assisted development workflows

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Project Structure

- App source lives at the repository root
- Main React entry: `src/main.tsx`
- Sections/components: `src/sections/` and `src/components/`

## Getting Started

From the repository root:

```bash
npm install
npm run dev
```

## Available Scripts

Run these from the repository root:

- `npm run dev` - Start local development server
- `npm run build` - Type-check and build for production
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint
- `npm run format` - Format source files with Prettier

## Notes

As this is both a portfolio and a learning experiment, the project may evolve quickly as I iterate on design, content, and AI-assisted coding practices.
